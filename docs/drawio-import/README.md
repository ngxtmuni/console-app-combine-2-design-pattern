# draw.io Import Guide

These Mermaid files can be imported directly into diagrams.net / draw.io.

Files:
- use-case-diagram.mmd
- class-diagram.mmd

How to import:
1. Open draw.io.
2. Choose Insert -> Advanced -> Mermaid.
3. Paste the content of one `.mmd` file.
4. Click Insert, then ungroup or edit shapes as needed.

Note:
- These versions are intentionally simple for presentation.
- The use-case diagram is represented as a Mermaid flowchart because this imports reliably in draw.io.
- The class diagram uses Mermaid `classDiagram` syntax.
